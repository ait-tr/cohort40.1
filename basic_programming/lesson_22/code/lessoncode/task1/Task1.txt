Написать методы, которые будут:
1) добавлять в массив сортов чая разные элементы
2) сортировать чай по параметру, который выберет пользователь
3) найти среди чая нужный сорт - есть ли он или нет

===================================================
Для решения 1 части задания необходимо:

- класс "Чай" с полями производитель, сорт
- класс "Производитель" с полями "Наименование" (в последующем добавим список всех видов чая этого производителя)
- класс "Все сорта чая", где будет храниться массив "Все чаи" и в этом классе
  будут все необходимые методы для работы с массивом

  ОПЕРАЦИИ С МАССИВАМИ (СТРУКТУРАМИ ДАННЫХ)
  CRUD
  - create ( создание добавление нового элемента в массив)
  - read (чтение, чаще задача поиска элемента в массиве)
  - update (обновление данных)
  - delete (удаление элемента)

  методы : add, findById, findByName, update (позже), delete
