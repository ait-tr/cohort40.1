empty() - boolean - true если стек пустой
pop() - "вытаскивает" элемент с вершины стека, т.е. из самой коллекции этот элемент удаляется
peek() - возвращает верхний элемент стека, но не удаляет его из коллекции
push(element) - помещает элемент в коллекцию ("вталкивает")
search(element)  - поиск элемента в стеке - есле элемент найден, то возвращается int - смещение от вершины стека
если не найден то -1
