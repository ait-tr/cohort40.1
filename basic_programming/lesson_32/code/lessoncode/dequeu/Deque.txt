addFirst(element) - добавление элемента в голову д/н очереди
addLast(element) - добавление элемента в хвост д/н очереди

E getFirst() - возвращает первый элемент (из головы) -элемент НЕ УДАЛЯЕТСЯ
E getLast() - возвращает последний элемент (из головы) -элемент НЕ УДАЛЯЕТСЯ

E pop() - возвращает ("вытаскивает") элемент из головы
push(E) - добавляет элемент в голову

E removeFirst()
E removeLast()

если элементов в очереди нет - то возвращается исключение IllegalStateException


-=======

E peekFirst() - возвращает элемент из головы
E peekLast() - возвращает элемент из хвоста

E pollFirst()
E pollLast() - возвращает элемент из головы / хвоста - удаляет его из очереди

если элементов в очереди нет - то возвращается null!!!
