В нашей текстовой последовательности присутствуют символы:
{, [, (, }, ], )

Правильной последовательностью является:
[]
[()]
{[()]}

1 сценарий:
- проходим по нашему набору символов
- если символом является ЛЮБАЯ открывающаяся скобка, то мы добавляем ее в коллекцию
- если символом является ЛЮБАЯ закрывающаяся скобка,то мы должны проверить
является ли эта закрывающаяся скобка парой для элемента который лежит на вершине стека

если "да", то вск ОК - мы должны вершину стека удалить (вытащить) и проверять дальше
если "нет" - то последовательность - некорректная

Дополнительные проверки:

- проверка на то, что в нашей текстовой последовательности не четное количество скобок

а)
- при обнаружении закрывающейся скобки - стек может быть пустой ->
это означает что количество закрывающихся скобок больше

б)- при завершении строки - стек может быть НЕ пустой ->
  это означает что количество открывающихся скобок больше
