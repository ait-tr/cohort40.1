реализовать простую лотерею использую конструкцию if-else-if
Т.З
1. создать класс для лотереи (подумать над названием)
2. добавить переменные для выигрышных номеров (5 номеров) (подумать над названием метода)
3. ограничить игровые номера от 1 до 50
4. создать метод, который рандомно сетит (добавляет) в переменные выигрышные номера
5. создать метод, который принимает 5 номеров в аргументах и проверяет их на выигрыш, и печатает все выигранные номера
6. в отдельном классе Test, в методе main протестировать работу лотереи.
(можно использовать Сканнер для ввода или хардкодом(самим) передать номера в метод лотереи)
