put() - добавить в коллекцию элемент (перезаписать)
get(ключ)  -получить данные из коллекции по ключу
size() - возвращает размер коллекции
isEmpty() - true/false - пуста ли наша коллекция
keySet() - возвращает коллекцию (Set) - которая состоит из всех ключей нашей коллекции

==================

containsKey() - проверяет если ли такой ключ в нашей коллекции
containsValue() - проверяет если ли такое значение в нашей коллекции
remove(объект)- удаление
