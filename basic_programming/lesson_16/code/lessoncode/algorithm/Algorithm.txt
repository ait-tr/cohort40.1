Сложности алгоритмов:

- логарифмическая - O(log(N)) - бинарный поиск
- N * log(N)                  - определенные алгоритмы сортировки
- N * N                       - двойные циклы (от 0 до N)
- N * 3
- Exp(N)
- N!
