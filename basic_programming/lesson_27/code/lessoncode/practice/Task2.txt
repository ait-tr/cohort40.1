Задача: Управление данными об образовательных курсах
Цели:

    Создать коллекцию для хранения данных о студентах. Студент имеет уникальный студенческий номер.
    Создать коллекцию для хранения данных о преподавателях. Преподаватель имеет уникальный идентификатор.
    Создать коллекцию для хранения данных о курсах. Каждый курс связан с одним преподавателем и может иметь несколько студентов.
    Добавить возможность назначения студентов на курсы.
    Создать коллекцию для хранения данных об отделениях (факультетах). Каждый курс принадлежит одному отделению.

Описание сущностей:

    Студент имеет следующие атрибуты: студенческий номер (уникальный), имя, список курсов.
    Преподаватель имеет следующие атрибуты: идентификатор (уникальный), имя, список курсов, которые он ведет.
    Курс включает в себя: название курса, идентификатор курса (уникальный), преподавателя, список студентов, отделение.
    Отделение (Факультет) содержит: название отделения, список курсов, принадлежащих этому отделению.

Задачи:

    Создайте и инициализируйте коллекции для студентов, преподавателей, курсов и отделений.
    Добавьте функциональность для назначения студентов на курсы и привязки курсов к преподавателям и отделениям.
    Создайте методы для вывода следующей информации:
        Список всех студентов, записанных на конкретный курс.
        Список всех курсов, которые ведет конкретный преподаватель.
        Список всех курсов в конкретном отделении.
