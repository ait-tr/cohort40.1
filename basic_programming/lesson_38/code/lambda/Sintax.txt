Лямбда оператор:   -> (некоторые трактуют как "становится" / "переходит")

код лямбда выражения может формироваться одним их двух способов:
1) одиночное выражение (одиночная лямбда)
2) блочное выражение - заключенные в {} - это случай когда в правой части нужно указать
несколько операторов и обязательно указывать return


(list_of_parameters) -> {
// тело лямбды
// ....
//....
return result;
}

list_of_parameters - список параметров, которое получает лямбда



Если в лямбда-выражении содержится единственный параметр, то можно без ()

i -> {
//...
//...
return j}


Возможно что лямбда не получает параметров
() -> {
//...
//...
return j}
