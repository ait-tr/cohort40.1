Задача:
Создайте систему для вычисления площадей различных
геометрических фигур, используя принципы ООП и обобщения (дженерики)
в Java. Система должна быть гибкой, чтобы поддерживать
различные типы чисел для задания размеров фигур (например, целые числа,
числа с плавающей точкой).

Вам необходимо:

    Определить интерфейс Figure, который будет содержать метод area(),
    возвращающий площадь фигуры.

    Реализовать класс Circle, который реализует интерфейс Figure и
    представляет круг. Класс Circle должен быть обобщенным, чтобы
    принимать радиус различных числовых типов. Метод area() для круга
    должен вычислять площадь как π×радиус^2.

    Реализовать класс Rectangle, который также реализует интерфейс
    Figure и представляет прямоугольник. Класс Rectangle должен быть
    обобщенным, принимая длины сторон прямоугольника в различных числовых типах.
    Площадь прямоугольника должна вычисляться как значение произведения сторон.

    В классе FigureDemo продемонстрируйте использование этих классов
    для вычисления площадей круга и прямоугольника, используя различные
    числовые типы для задания их размеров.
