Напишите программу, которая должна имитировать раздачу карт для игры в покер.
Программа получает число n, задаваемое с консоли пользователем,
и раздает карты на n игроков (по 5 карт каждому) из рассортированной колоды.
Выведите информацию о картах для каждого игрока.
Разделяйте пять карт, выданных каждому игроку, пустой строкой.

=================================================================

Описание нашего приложения:

- класс карта
- массив карт (рассортирован - перемешаны)
- класс карточная колода в котором будет хранится массив карт
- класс игрок у которого будет массив из 5 карт
- класс карточный стол в котором будет массив игроков


Запуск приложения:
- подготовительные действия:
    - создать карточный стол
    - создать карты
    - создать колоду (наполнить массив необходимым количеством экземпляров класса карта)
    - перемешать колоду
- спросить у пользователя сколько будет игроков
- создать необходимое количество игроков и поместить их массив игроков
- раздать карты игрокам
- вывести информацию о картах у игроков


колода карт - массив из 52 карт (4 масти по 13 карт от 2 до туза)
