Написать приложение, которое демонстрирует игру "Лотерея"

1) Для игрока выдается три лотерейных билета, который содержит 5 чисел
2) Игроков может быть несколько
3) Производится розыгрыш лотереи
4) Выпадают пять случайных значений, которые не должны повторяться.
5) Диапазон значений от 1 до 36


--------------------------------------------------------------

- игрок
- массив для хранения значений лотерейного билета -> создать класс Lottery ticket
- этих экземпляров классов будет три
- массив для хранения ВСЕХ возможных номеров из которых мы будем случайным образом получать значения -> класс Lottery
- массив для хранения выигрышных номеров
- PlayLottery - класс для проведения лотереи

- добавить возможность хранения истории розыгрышей
