toList, toSet, toMap

    toList(): собирает элементы потока в List. Порядок элементов в списке соответствует порядку их следования в потоке.
    toSet(): собирает элементы потока в Set. Это удаляет любые дублирующиеся значения, поскольку множество не может содержать дубликаты.
    toMap(Function keyMapper, Function valueMapper): создает Map из элементов потока, где каждый элемент преобразуется в ключ и значение с помощью предоставленных функций. Если поток содержит дублирующиеся значения, которые мапятся в один и тот же ключ, будет выброшено исключение IllegalStateException.

groupingBy

    groupingBy(Function classifier): группирует элементы потока по критерию, заданному функцией классификатором, и собирает данные в Map, где ключ — это результат функции классификатора, а значение — это список элементов, соответствующих этому ключу.
    groupingBy(Function classifier, Collector downstream): похож на предыдущий, но добавляет возможность использовать другой коллектор для результатов каждой группы, что позволяет выполнять дополнительные преобразования или агрегации.

joining

    joining(): соединяет элементы потока в одну строку. Это особенно полезно для строковых потоков.
    joining(CharSequence delimiter): соединяет элементы, используя разделитель.

reducing

    reducing(BinaryOperator op): выполняет редукцию элементов потока, используя ассоциативную и коммутативную функцию. Это может быть использовано для вычисления суммы, максимума, минимума и т.д.

mapping

    mapping(Function mapper, Collector downstream): применяет функцию преобразования к каждому элементу, а затем собирает результаты с использованием другого коллектора. Это полезно, когда необходимо преобразовать элементы перед их сборкой.

count, summingInt, averagingInt и другие

    count(): подсчитывает количество элементов в потоке.
    summingInt(ToIntFunction mapper): суммирует значения, предоставляемые функцией, применяемой к каждому элементу.
    averagingInt(ToIntFunction mapper): вычисляет среднее значение для значений, предоставляемых функцией, применяемой к элементам.
