Написать приложение для учета и отражения операций с различными медиа-материалами:
книги, журналы, и тд.

Базовая функциональность:
- добавить новый носитель
- отобразить информацию о доступных материалах

===============================================

Класс - MediaItem - общий класс для всех медиа-материалов с базовыми свойствами и методами

- название
- автор
- год издания

Класс Book

- количество страниц


Класс Magazine

- номер выпуска


Класс LibraryCatalog
