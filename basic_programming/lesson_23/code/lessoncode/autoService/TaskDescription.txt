Написать приложение для автомастерской со следующим функционалом:
- хранить информацию об автомобилях разного типа
- добавлять новые
- иметь возможность"назначить" автомобилю определенный вид сервисных работ (ТО, ремонт, диагностика и тд)
- контролировать статус исполнения заказа

===================================================

- Класс Vehicle - абстрактный класс для общих характеристик (VIN, марка, модель и год выпуска)

- Интерфейс Serviceable - методы для назначения работ (ТО, ремонт, диагностика и тд)

- Три класса для различного вида работ
 - Repair - ремонт
 - Painting - покраска
 - Maintenance -

 - описание работ
 - стоимость
 - ожидаемое время выполнения

 - ServiceManager - для создания массивов и работы с ними

 - ServiceApp
