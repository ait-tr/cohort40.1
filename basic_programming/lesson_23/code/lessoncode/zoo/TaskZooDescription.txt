Разработать систему для учета и реализации действий животных в зоопарке.
Различные типы животных: млекопитающие, птицы и рептилии.
Каждый класс будет иметь общие характеристики и свои уникальные.
Нужно уметь добавлять новых животных, обрабатывать их поведение и
выводить о них информацию.

============================

Класс Animal - для базовых характеристик (имя и возраст)
+ метод для звука
+ метод для хождения ("ходить")

Класс Mammal - млекопитающие

Класс Bird - птицы

Класс Reptile = рептилии

интерфейс Flyable с методом fly() который будет реализован в классе Bird


Класс для хранения всех животных (массив) и методы доступа к нему
