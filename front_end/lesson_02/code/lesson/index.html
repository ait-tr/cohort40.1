Базовые консольные команды гита

git clone тут_должен_буть_путь_к_удаленному_репозитарию Клонировать репозитарий

git status посмотреть изменения

git add . добавить изменения в индекс

git commit -m 'описание коммита' закоммитить изменения (сохранить локально)

git push отправить закомиченные изменения на удаленный репозитарий

git pull подтянуть изменения из удаленного репозитария к себе на компьютер
